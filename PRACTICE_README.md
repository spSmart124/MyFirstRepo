# My Project about markdown.
## Header
### Syntax
You can use six levels of header
* # Header 1
* ## Header 2
* ### Header 3
* #### Header 4
* ##### Header 5
* ###### Header 6

You can also use === and --- to represent a header.

### Example
My Header
===

My Header 1
---
